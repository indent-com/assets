# Indent Assets

Brand assets and logos for Indent.

## Contents

This repository contains official Indent logos in multiple formats and color variations:

### Logo Variations

- **Green Indent Logo** - `green_indent_logo.{gif,png,svg}`
- **Red Indent Logo** - `red_indent_logo.{gif,png,svg}`
- **Yellow Indent Logo** - `yellow_indent_logo.{gif,png,svg}`
- **Grey Indent Logo** - `indent-logo-grey.png`
- **Animated Spin** - `indent_spin.gif`

### Formats

- **SVG** - Vector format, scalable to any size
- **PNG** - Raster format with transparency
- **GIF** - Animated format

## Usage

These assets are intended for use in Indent-related projects, documentation, and marketing materials.
